# 平台控制台与应用数据源 实施计划

## 架构流转

```mermaid
flowchart TD
  Login["/login 登录页"] -->|登录成功| Console["/console 平台控制台\nConsoleLayout - 纯顶部导航"]
  Console -->|点击应用卡片| Workspace["/apps/:appId 应用工作台\nAppWorkspaceLayout - 左侧边栏"]
  Console --> DS["/console/datasources\n平台数据源管理"]
  Console --> Migration["/console/migration\n数据迁移中心(Phase5预留)"]
  Workspace --> Dashboard["/apps/:appId/dashboard"]
  Workspace --> Forms["/apps/:appId/forms"]
  Workspace --> Builder["/apps/:appId/builder"]
  Workspace --> Settings["/apps/:appId/settings"]
  Settings --> DSPage["/settings/datasource 只读+测试"]
  Settings --> Sharing["/settings/sharing 共享策略"]
  Settings --> Aliases["/settings/aliases 实体别名"]
```



## 数据模型依赖

```mermaid
erDiagram
  LowCodeApp {
    long DataSourceId "FK, nullable, 创建后不可改"
    bool UseSharedUsers "default true"
    bool UseSharedRoles "default true"
    bool UseSharedDepartments "default true"
  }
  TenantDataSource {
    long AppId "nullable, null=平台级"
    int MaxPoolSize "default 100"
    int ConnectionTimeoutSeconds "default 30"
    bool LastTestSuccess "nullable"
    DateTimeOffset LastTestedAt "nullable"
  }
  AppEntityAlias {
    long AppId "FK"
    string EntityType "user/role/department"
    string SingularAlias
    string PluralAlias "nullable"
  }
  LowCodeApp ||--o| TenantDataSource : "DataSourceId"
  LowCodeApp ||--o{ AppEntityAlias : "AppId"
```

